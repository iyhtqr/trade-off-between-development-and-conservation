# trade-off-between-development-and-conservation
An interactive physics‑based simulator exploring the tension between urban development and heritage/greenery conservation in Wong Tai Sin. 
Users add “development towers” and “heritage/greenery blocks” to a shared 3D plaza. 
As more towers are added, they physically push on heritage elements, increasing “development pressure” and lowering the social‑equity score. 
The goal is to experiment with different combinations: too much development destabilizes the space, while a balanced mix of towers and preserved greenery keeps the system more stable.
